# Sorting Algorithms

This directory contains implementations of various sorting algorithms in multiple languages.

## Algorithms

- [QuickSort](./quicksort/) - Divide and conquer sorting algorithm

## Adding New Algorithms

When adding a new sorting algorithm:
1. Create a new subdirectory with the algorithm name (e.g., `mergesort/`)
2. Create language subfolders: `java/`, `python/`, `cpp/`, `c/`
3. Add a README.md in the algorithm directory
4. Update this README with a link to the new algorithm
