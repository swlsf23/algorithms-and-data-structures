# Linked List

This directory contains implementations of a linked list data structure in multiple languages.

## Implementations

- [Java](./java/)
- [Python](./python/)
- [C++](./cpp/)
- [C](./c/)

## Operations

- Insert
- Delete
- Search
- Traverse

## Complexity

| Operation | Time Complexity | Space Complexity |
|-----------|-----------------|------------------|
| Insert | O(1) | O(1) |
| Delete | O(n) | O(1) |
| Search | O(n) | O(1) |
| Traverse | O(n) | O(1) |
