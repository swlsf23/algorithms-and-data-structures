# Binary Tree

This directory contains implementations of a binary tree data structure in multiple languages.

## Implementations

- [Java](./java/)
- [Python](./python/)
- [C++](./cpp/)
- [C](./c/)

## Operations

- Insert
- Delete
- Search
- Traversal (In-order, Pre-order, Post-order)

## Complexity

| Operation | Time (Average) | Time (Worst) | Space |
|-----------|----------------|--------------|-------|
| Insert | O(log n) | O(n) | O(1) |
| Delete | O(log n) | O(n) | O(1) |
| Search | O(log n) | O(n) | O(1) |
| Traverse | O(n) | O(n) | O(h) |
